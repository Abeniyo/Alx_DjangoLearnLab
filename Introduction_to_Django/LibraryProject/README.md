# LibraryProject

This is the **Introduction to Django** project for the ALX Back-End Web Development program.

## Project Overview
This project introduces the basics of Django by:
- Setting up a Django development environment
- Creating a Django project and app
- Defining models and using Django ORM
- Performing CRUD operations
- Using the Django admin interface

## Technologies Used
- Python
- Django
